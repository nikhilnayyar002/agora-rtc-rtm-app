# Agora Live + RTM (Custom)

## Instructions

* Installation
    ```
    npm ci
    ```
### Client

```src/*.*``` files. Compliled files are stored in ```dist/*.*```.

* Hot Reload
    ```
    npm run watch
    ```
    
* Build
    ```
    npm run build
    ```

### Server

```server.js``` file.

* Hot Reload
    ```
    npm run server
    ```
